==============
Project Description
==============

Mellorine Team would like to bring back Text adventure games!. Textadventure game were the earliest type of computer game, from a time when computers could only display text - there were no graphics, so everything was described with text. 

You could play the game by typing commands with the keyboard such as "north" or "hit" depending on the given menu.
